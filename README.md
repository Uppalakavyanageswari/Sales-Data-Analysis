# Sales Data Analysis Project

## Project Overview

This project analyzes retail sales data using Python, SQL, and Power BI to discover business insights and build an interactive dashboard.

## Tools Used

* Python
* Pandas
* Matplotlib
* SQL (SQLite)
* Microsoft Excel
* Power BI

## Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. SQL Analysis
4. Power BI Dashboard
5. Business Insights

## Dashboard Features

* Total Sales KPI
* Total Orders KPI
* Average Sales KPI
* Sales by Category
* Sales by Region
* Monthly Sales Trend
* Top 10 Products
* Sales by Segment
* Interactive Slicers

## Business Insights

* Identified the highest-selling products.
* Compared sales across regions and categories.
* Analyzed monthly sales trends.
* Built an interactive dashboard for business decision-making.
